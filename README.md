# StockLite -股基金持仓管理工具

一个基于Electron + React + TypeScript的轻量级桌面金融工具，专注于股票和基金的持仓收益管理。

##🎯 产品特性

- **轻量快速** - 启动迅速，界面简洁
- **专注收益** - 专为持仓收益计算设计
- **分组管理** -支持自定义分组和组织
- **实时更新** -行情自动实时刷新
- **本地存储** - 数据安全存储在本地
- **深色模式** -支持深色/浅色主题切换

##🚀技术栈

- **Electron 40.6.1** -应用框架
- **React 19.2.4** -前端UI库
- **TypeScript 5.9.3** - 类型安全
- **Vite 7.3.1** -构建工具
- **Zustand 5.0.11** - 状态管理
- **better-sqlite3 12.6.2** - 本地数据库
- **Tailwind CSS 4.2.1** -样式框架
- **Lucide React** - 图标库

## 📊 数据结构

###分组表 (stock_groups)
```sql
CREATE TABLE stock_groups (
  id TEXT PRIMARY KEY,
  name TEXT NOT NULL,
  created_at INTEGER NOT NULL
)
```

###持仓表 (stocks)
```sql
CREATE TABLE stocks (
  id TEXT PRIMARY KEY,
  symbol TEXT NOT NULL,
  name TEXT NOT NULL,
  group_id TEXT NOT NULL,
  cost_price REAL NOT NULL,
  quantity REAL NOT NULL,
  created_at INTEGER NOT NULL
)
```

###收计算公式
```
持仓成本 = 成本价 ×持数量
当前市值 = 当前价格 ×持数量
浮动收益 = 当前市值 - 持仓成本
收益率 =浮收益 / 持仓成本
```

##🎨 界面预览

应用采用现代化的UI设计：
- 顶部导航栏：应用标题、Tab切换、刷新控制、主题切换
-左分组面板：分组管理、新建分组
-右内容区域：持仓列表、收益展示

##🔧配置说明

###刷新设置
-刷新间隔：5秒（默认）
-基刷新间隔：60秒（默认）
-支持全局刷新开关控制

### 数据库
- 数据库存储路径：`%APPDATA%/stocklite.db`
- 使用SQLite本地存储，数据安全可靠

## 📝 使用说明

1. **创建分组**：在左侧分组面板中创建新的股票或基金分组
2. **添加持仓**：在右侧内容区域填写股票信息（代码、名称、成本价、数量）
3. **查看收益**：系统自动计算并显示持仓收益和收益率
4. **管理刷新**：通过顶部工具栏控制自动刷新开关
5. **切换主题**：使用太阳/月亮图标切换深色/浅色模式
